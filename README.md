# ふくらむ先生

小学2年生が「文章の膨らませ方」を自然に身につけるための、タブレット向け作文ゲームです。
国語の教科書（光村図書2年「丸、点、かぎ」）の単元と関連づけて設計しています。

**ファイル1つ・インストール不要・オフラインで動作**します。`index.html` をブラウザで開くだけです。

## 🎮 遊んでみる

👉 **[ここをタップして遊ぶ](https://happy-field-edu.github.io/fukuramu-sensei/)**

（タブレットのブラウザ推奨。ネット接続がなくても、一度開けば動きます）

## こんな子に

- 「早く終わらせること」に価値を感じていて、作文が二文で終わってしまう
- 「はじめ・中・おわり」の "中" がスカスカ
- 気持ちの言葉が「たのしかったです。」だけになりがち
- 色・大きさ・形・数などの情報を付け足すのが苦手

## 5つのモード

| モード | ねらい |
|---|---|
| 🎈 **ぶんしょう ふくらませ** | 「だれと？」「どんな色？」「なんかい？」の質問カードに答えるたびに、文章と風船が目の前でふくらむ。付け足した部分が色分けで見える。 |
| ✏️ **まる・てん・かぎ めいじん** | 教科書「丸、点、かぎ」準拠の3ステージ。①記号置き ②点の位置で意味が変わる体験 ③縦書きマス目に記号を書く。全クリアで認定証。 |
| 👓 **じぶんの ぶんで ちょうせん** | 子どもが書いた作文をその場で読み取り、「ふくらむ先生」が褒めて・足りないところを提案。**サーバー不要・完全オフライン**の日本語解析。 |
| 💗 **きもちことば あつめ** | 「たのしかった」以外の気持ちの言葉を、場面クイズで集める。 |
| 📖 **きもちことば ずかん** | 集めた言葉を図鑑で確認。作文モードのヒントとも連動。 |

## 設計のポイント

- ⏱ **タイマーなし**。得点は「ふくらませた数」だけ ── 早く終わらせても得しない
- 🔤 **ほぼひらがな＋全漢字にふりがな**（ruby）
- 📱 タブレット縦持ちで押しやすい大きめボタン
- 🔊 音声読み上げ対応（音読が苦手な子の支え）
- 🗂 進捗はブラウザ内（localStorage）に保存。個人情報の送信は一切なし

## 教科書との対応（光村図書2年「丸、点、かぎ」p.89）

- 「このみせで、はきものをかう／このみせでは、きものをかう」など、**点の位置で意味が変わる**例をそのままゲーム化
- 「せんせいにきいてみたら、『それは、さるすべりのきですよ。』」の**マス目書き**を再現
- 「会話は行をかえて書く」「丸・点・かぎは一ますに書く」ルールを操作で体験

## 使い方（先生向け）

1. `index.html` をタブレットのブラウザで開く（Google Drive等に置いてリンクを配ってもOK）
2. 授業の流れの例：「②点クイズ → 教科書 p.89 → ③マス目」→ 最後に「④じぶんの ぶんで ちょうせん」で自分の作文を実戦
3. ネット環境がなくても動作します
